## Para rodar em local:


**Instalar dependências**

```bash
yarn install
```

**Acessar diretório**

```bash
cd qr-code-generator
```

**Iniciar Servidor de desenvolvimento**
```bash
yarn start
```

```
Acesse o endereço http://localhost:3000
```
